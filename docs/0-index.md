# An Introduction to Mapping and Spatial Analysis with QGIS

<b>QGIS</b> is a fully-featured desktop geographic information systems (GIS)
application used by public and private
organizations worldwide. <b>Unlike ArcGIS, it is free/open source, and cross platform
(meaning it will run on Windows, MacOS, or Linux)</b>. QGIS is actively maintained
by a team of volunteer developers and its functionality can be extended with the
use of custom plugins.

After following the examples in this tutorial, you'll understand some common
QGIS workflows that can be adapted to support your own research or coursework.
This tutorial is designed for learners from all backgrounds; you don't need any
previous experience with GIS or statistics to benefit from these materials.


<b>You can download the latest version of QGIS for free from
<a href="https://www.qgis.org" target="_blank">qgis.org</a></b>.
It has also been installed on
<a href="https://mdl.library.utoronto.ca/technology/computers-with-gis-software" target="_blank">
lab computers in several of UofT's libraries.</a>
